Titans Hub — Gateway Pix
Sem chaves

As chaves são enviadas ao servidor proxy e nunca ficam expostas no frontend. O checkout chama POST /api/pix/create que faz a requisição autenticada.

O evento Purchase é enviado a todos os Pixels quando o Pix é pago. Eventos de PageView/AddToCart não são enviados para evitar poluição no seu gerenciador de anúncios.
A conversão é disparada em todas as Tags quando o Pix é pago. Apenas vendas reais disparam a conversão.
